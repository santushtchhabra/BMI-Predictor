About This Project

This project predicts Body Mass Index (BMI) using Decision Tree models. It uses Flask as backend, TailwindCSS for UI, and Matplotlib/Seaborn for visualization.

  • Decision Tree Regressor → Predict BMI Value
  • Decision Tree Classifier → Predict BMI Category
  • Dataset: 500 Person Gender, Height, Weight, BMI
